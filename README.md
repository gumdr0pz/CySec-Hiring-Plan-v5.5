## Overview

This repository contains the complete **v5.5** hiring plan for the Cybersecurity Business Unit (BU): **20 positions** across **9 teams** over 12 months, in a **support-pod 7-5-8** wave structure. Each wave builds deliverable capability, pairing seniors with junior capacity where it scales.

**📦 What's Current:**

- ✅ Wave 1: 7 (4 Senior + 3 Junior) | Wave 2: 5 (Senior/Lead) | Wave 3: 8 (1 Senior + 7 Junior/IC)

## COGS Formula

```
Burdened Cost = Base Salary × 1.44x
```

**Components:** COLA 10% · EPF 11% · SOCSO 1.45% · Health RM100/mo · Bonus 10% · PTO 12 hrs

**Salary Bands (30% reduction applied):**
- Senior / Lead: RM4,900–7,840/mo base → RM84,672–135,480/yr burdened
- Junior / IC / Junior-Intermediate: RM3,500–5,600/mo base → RM60,480–96,768/yr burdened
