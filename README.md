## COGS Formula

Based on MSP_Cost_Calculator.xlsx (TECH Cost--Burdened sheet):

```
Burdened Cost = Base Salary × 1.44x
```

**Multiplier Components:**
- COLA: 10%
- EPF: 11%
- SOCSO: 1.45%
- Health Insurance: RM100/month
- Performance Bonus: 10%
- PTO Accrual: 12 hours
