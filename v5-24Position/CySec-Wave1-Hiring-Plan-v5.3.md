## DETAILED POSITION DESCRIPTIONS

*(Note: Full JDs are in the JDs/ folder. Summaries below.)*

### 3.1 Gov Sec PM (Senior) - Cybersecurity
- **Focus:** Government security compliance and governance
- **Key Skills:** CISSP/CISM, Malaysian gov frameworks, stakeholder management
- **Success:** Security governance framework, first gov engagement

### 3.2 Sr GRC Specialist (Senior) - VORON
- **Focus:** GRC platform operations, risk management, compliance
- **Key Skills:** VoronCitadel platform, regulatory requirements, control frameworks
- **Success:** GRC platform deployed, first customer onboarded

### 3.3 Sr DevSecOps (Senior) - DevSecOps
- **Focus:** Security automation, CI/CD pipelines, cloud security
- **Key Skills:** CI/CD tools, cloud platforms, scripting (Python/Bash)
- **Success:** DevSecOps pipeline implemented, security monitoring established

### 3.4 PM (Senior) - PM Team
- **Focus:** Project delivery, stakeholder coordination, process improvement
- **Key Skills:** PMP/PRINCE2, Agile/Scrum, stakeholder management
- **Success:** PM methodology established, first project portfolio

### 3.5 Sr PS (Senior) - Professional Services
- **Focus:** Service delivery, technical implementation, client relationships
- **Key Skills:** Cybersecurity/blockchain technical background, client management
- **Success:** Service delivery framework, first client engagement

### 3.6 Sales Specialist (Enterprise) (Senior) - Sales
- **Focus:** Enterprise sales, account management, pipeline development
- **Key Skills:** Enterprise sales experience, cybersecurity knowledge, negotiation
- **Success:** RM2M+ pipeline, first enterprise deal closed

### 3.7 Presales Consultant (Senior) - Sales
- **Focus:** Technical presales, solution design, demos/POCs
- **Key Skills:** Solution engineering, presentations, RFP/RFI responses
- **Success:** Presales process established, demo environment configured
